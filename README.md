# Claude Code Marketplace

## Requirements


### Install [Claude Code](https://claude.com/product/claude-code)

- **CLI Installation**

Copy and paste the following command into your terminal:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

- **[Claude Code for VS Code](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code&ssr=false#review-details)**

## Installation

1. Open Claude code

- **Using Terminal**

```bash
claude
```

- **Using VSCode**

***Click*** on the claude code icon in the top right corner of the screen.

![CC Icon to Open](./assets/vscode/cc-icon-to-open.png)

2. Add the local marketplace

```bash
/plugin marketplace add yersonargotev/cc
```

3. Install the plugins

```bash
/plugin install cc@yersonargotev/cc
```
